# Sample Vault

A small example vault for trying out HelloNotes. Open this folder with
**Select Vault Folder** in the app.

Each note demonstrates a feature:

| Note | Shows off |
|------|-----------|
| Welcome.md | Front matter (hidden, shown in the Properties panel), aliases, headings |
| Ideas.md | Wiki-links, heading links, and hashtags |
| Projects/Roadmap.md | Folders / nested notes |
| Callouts.md | Callouts (with collapsible folds) and hidden comments |
| Diagram.md | Inline Mermaid diagrams |
| MathNote.md | Inline and block LaTeX, plus a Mermaid flowchart |
| Transclude.md | Note transclusion — embedding a whole note or one section |
| RichTransclude.md | Transclusion of a note containing math and a diagram |
| 2026-07-11.md | A daily note |
| Templates/Daily.md | A note template |

> This is demo content — feel free to edit or delete anything.
